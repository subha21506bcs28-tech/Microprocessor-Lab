# EX1 – Implementation of Arithmetic Operations in 8086

##  Aim
To implement basic arithmetic operations using 8086 assembly language.

##  Programs Included
1. Addition of two 16-bit numbers
2. Subtraction of two 16-bit numbers
3. Multiplication of two 16-bit numbers
4. Division of two 16-bit numbers
5. Evaluation of expression (a² + b² + c²)

## Concepts Used
- Data Transfer Instructions (MOV)
- Arithmetic Instructions (ADD, SUB, MUL, DIV)
- Flag Handling (Carry, Borrow)
- Register Usage (AX, BX, DX)
- Memory Storage

##  Working Principle
- Arithmetic operations are performed using registers such as AX, BX, and DX.
- Carry and borrow conditions are checked using flags.
- Results are stored in memory locations.
- For multiplication and division, AX and DX registers are used to store results.

##  Outcome
- Understanding of arithmetic operations in 8086
- Knowledge of handling carry and borrow
- Ability to evaluate mathematical expressions
- Familiarity with register-level programming
