## EX-3 
## Aim
To implement array manipulation techniques such as sorting, searching, and reversing using 8086 assembly language.

## Programs Included

### 1. Sorting (Ascending Order)
- Sorts an array using Bubble Sort technique
- Compares adjacent elements and swaps if required

### 2. Searching (Linear Search)
- Searches for a given key element in an array
- Counts occurrences of the element

### 3. Reverse Array
- Reverses the elements of an array
- Uses two-pointer method (start and end swapping)

## 🛠️ Concepts Used
- Array Processing
- Looping and Iteration
- Conditional Instructions (CMP, JNZ, JGE)
- Data Exchange (XCHG)
- Pointer Manipulation (SI, DI)

##  Outcome
- Understanding of array operations in assembly language
- Knowledge of sorting and searching techniques
- Ability to manipulate memory efficiently
