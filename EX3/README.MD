# EX3 – 8086 Assembly Language Programs
## Aim
To study and implement basic 8086 assembly language programs.
## Programs Included
1. Display character 'I' using BIOS interrupt
2. HELLO WORLD using file handling
3. String comparison
4. Display pattern 'C'
## Concepts Used
- BIOS Interrupt (INT 10H)
- DOS Interrupt (INT 21H)
- String Instructions (CMPSB)
- File Handling
## Outcome
Understanding of basic input/output operations, string handling, and interrupt usage in 8086.
