# EX6 – Stepper Motor Control using 8255

##  Aim
To control the rotation of a stepper motor using 8255 Programmable Peripheral Interface.

## Programs Included
1. 360° Rotation
2. 90° Rotation (Forward and Reverse)

## Concepts Used
- 8255 Interfacing
- Stepper Motor Control
- Sequence Generation
- Delay Techniques

##  Working Principle
The stepper motor rotates based on input pulse sequences. By sending proper bit patterns through 8255 ports, the motor rotates step-by-step in desired direction.

##  Outcome
- Understanding of stepper motor interfacing
- Ability to control direction and angle of rotation
- Knowledge of hardware control using 8255
