# EX9 – Data Conversion and Sorting using 8051

##  Aim
To perform data conversion and sorting operations using 8051 microcontroller.

##  Programs Included
1. Hexadecimal to ASCII Conversion
2. ASCII to Hexadecimal Conversion
3. BCD to Binary Conversion
4. Sorting of Array (Ascending Order)
5. Octal to hexadecimal conversion
##  Concepts Used
- 8051 Assembly Language
- External Memory Access (MOVX)
- Data Pointer (DPTR)
- Bit Masking (ANL)
- Nibble Operations (SWAP)
- Conditional Instructions (CJNE, JC)
- Looping (DJNZ)

##  Working Principle
- Data is read from external memory using DPTR.
- Conversion operations are performed using masking and arithmetic instructions.
- ASCII and Hex conversions use offset adjustments (30H, 07H).
- Sorting is done by comparing adjacent elements and swapping when required.
##  Outcome
- Understanding of number system conversions
- Knowledge of bit manipulation and masking
- Ability to implement sorting in assembly language
- Improved understanding of memory handling in 8051
