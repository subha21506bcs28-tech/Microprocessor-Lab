# EX5 – Waveform Generation using 8255

## Aim
To generate different waveforms using 8255 Programmable Peripheral Interface.

## Programs Included
1. ADC Interfacing
2. Square Wave Generation
3. Triangle Wave Generation
4. Staircase Wave Generation
5. Sine Wave Generation

## Concepts Used
- 8255 Interfacing
- Digital to Analog Concepts
- Looping and Delay Techniques
- Lookup Table Method

##  Working Principle
Different waveforms are generated by sending varying digital values through the output port of 8255. Delay is used to control the shape and frequency of the waveform.

## Outcome
- Understanding waveform generation techniques
- Knowledge of DAC/ADC interfacing
- Ability to generate analog-like signals using digital systems
