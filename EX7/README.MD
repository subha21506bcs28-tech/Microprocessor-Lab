# EX7 – 8255 Modes of Operation
## Aim
To study and implement different modes of operation of the 8255 Programmable Peripheral Interface (PPI).
## Programs Included
1. Mode 0 – Basic Input/Output Mode
2. Mode 2 – Bidirectional Data Transfer Mode
3. Mode 3 – Control/Extended Mode Operation
## Concepts Used
- 8255 Control Word Format
- Port Configuration (Port A, Port B, Port C)
- Input/Output Operations
- Mode Selection
## Outcome
Understanding the working of different modes of 8255 and how to configure ports for various interfacing applications.
